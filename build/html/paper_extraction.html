Text Extraction

The dataset has the provided OCR XMLs from which the data for the newspapers were extracted in the form of JSONs in which each JSON contains all the data for the single paper.

The data contains the important meta data like paper id, page number and text related to the page number. The extracted text was as per OCR XMLs hence it contains alot of lossy extraction.

The future plans is to use the OCR correction fto improve the texts.
