Text Extraction
==============================

**Text Extraction**

The text from the XMLs is extracted to store in systematic way in JSON,
The JSON contains the essential keys value pairs for example, issue date, text for the each pages, and overall text for the page.

After the extraction each, directory contains a JSON file which contains the extracted text for the future use of the text.
